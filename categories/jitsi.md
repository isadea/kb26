## What is {{variable.Jitsi}}?
{{variable.Jitsi}} is a video conferencing tool available on desktop and mobile. The {{variable.Jitsi}} integration is designed to let customers join virtual rooms to have video and audio meetings on the fly. {{variable.Jitsi}} virtual rooms are similar to Zoom meeting links that anyone with the link can join. 

:::(info) ({{variable.Jitsi}} [charges you**per user**](https://jaas.8x8.vc/#/plan/upgrade){target="_blank"} for this integration.)
:::
## How the {{variable.Jitsi}} integration works
You need a {{variable.Jitsi}} account, an API key dedicated to the integration and your {{variable.Jitsi}} app ID to set up the integration. Once you have all the relevant details you can enable the integration in a few clicks {{variable.OnTheAdminPanel}}. 

{{variable.MoreInformation}} [Integrating Jitsi by 8x8](/v3/docs/integrating-jitsi){target="_blank"}


As soon as you complete the integration, the relevant customers and teams will be able to use virtual rooms to connect with other customers.
